# Diminished Chords and Tritone Substitution as Chromatic Passing Devices

Diminished-seventh chords and tritone substitution both work by exploiting the same underlying property of the tritone interval — it splits an octave exactly in half, which means it resolves ambiguously and can substitute for or connect harmonies a tritone apart. This knowledge base documents both devices as the harmonic vocabulary that most sharply separates bebop and late-Romantic classical harmony from simpler diatonic genres.

## Diminished-Seventh Chords in Late-Romantic Chromaticism

`romantic.md` names the diminished-seventh chord alongside secondary dominants as one of the era's core chromatic-tension tools: "Extended chromatic harmony with frequent secondary dominants and diminished-seventh chords," used together with deceptive cadences to "delay and complicate resolution for maximum emotional effect." The diminished-seventh chord's symmetrical structure (stacked minor thirds) is what makes it so useful for this purpose — because it's built entirely from equal intervals, the same diminished-seventh chord can resolve convincingly to any of four different tonal centers, giving a composer maximum flexibility for redirecting harmony unexpectedly, which is exactly the "complicated resolution" effect `romantic.md` describes.

## Tritone Substitution in Bebop

`bebop.md` documents tritone substitution as a defining feature of the genre's harmonic vocabulary at the phrase level: "ii-V-I with tritone substitutions." The technique replaces a dominant chord with another dominant chord built a tritone away — because the two chords share the same tritone interval between their 3rd and 7th (just inverted), the substitute resolves with the same voice-leading logic as the original while producing a different root motion and a more chromatic, "slippery" bass line. `bebop.md`'s bass-programming guidance treats correctly navigating this vocabulary as a genre-defining skill in itself: "Program dense walking quarter-note lines that outline rapid ii-V-I chains, chromatic passing tones, and tritone substitutions accurately — bebop bass requires harmonically 'correct' note choices more than any earlier jazz style, so avoid generic root-fifth patterns."

## The Diminished Scale as Bebop's Corresponding Melodic Vocabulary

`bebop.md` also documents the diminished (half-whole) scale as the melodic partner to these harmonic devices: a scale built from alternating half and whole steps, symmetrical in the same way the diminished-seventh chord is, giving improvisers a scale that outlines dominant/diminished harmony's tension convincingly over both plain dominant chords and tritone-substituted ones. `jazz_fusion.md` documents the same scale (labeled "diminished (whole-half)," the complementary alternating-order variant) as part of its own extended harmonic-improvisation vocabulary, showing the device's persistence beyond bebop into later jazz-descended styles.

## Circle-of-Fifths Turnarounds With Substitute Dominants

`bossa_nova.md` documents tritone substitution used specifically within turnaround progressions: "Circle-of-fifths turnarounds with substitute dominants (tritone subs)." This shows the technique functioning at a different structural scale than bebop's phrase-level ii-V-I substitution — here it's a recurring device within the turnaround (the harmonic figure that cycles a tune back to its top), giving an otherwise-standard circle-of-fifths progression a more chromatic, jazz-colored character at a specific, recurring structural point rather than throughout the whole tune.

## Common Mistakes

**Using a diminished-seventh chord or tritone substitution without considering the alternate resolution it implies.** Because both devices work by exploiting harmonic ambiguity, using them "decoratively" without following through on the different resolution they set up (per `romantic.md`'s deceptive-cadence-adjacent logic) can read as an error rather than an intentional chromatic move.

**Treating tritone substitution as interchangeable with the original dominant in every context.** `bebop.md`'s emphasis on "harmonically correct" bass note choices implies the substitution changes what the surrounding melodic/bass vocabulary should outline — a bass line built for the original dominant doesn't automatically fit the substitute's altered root motion.
