# External Directory: ad-si/awesome-music-production

[ad-si/awesome-music-production](https://github.com/ad-si/awesome-music-production) is a community-curated "awesome list" of music-production software, hardware, and resources — a flat directory of external tools with one-line descriptions and links, not deep per-tool technique documentation like `knowledge_base/vst_database/`. This entry is a snapshot pointer, not a mirror: the linked repo updates independently of this project, so treat the live URL as the source of truth and refetch it rather than assuming this snapshot stays current.

## When to use this

Consult this doc (or refetch the live URL) when a producer or agent needs a tool — a DAW, sample library, MIDI controller, live-coding environment, AI music tool — that isn't documented in depth anywhere else in this knowledge base, before falling back to unsourced general knowledge. It's a starting point for discovering what exists, not a substitute for `knowledge_base/vst_database/`'s deeper technique entries once a specific tool is actually in use.

This project's `knowledge_base/daw/` deliberately covers only Ableton Live (primary) and FL Studio (secondary) per `CLAUDE.md`; this list is where to look if a project ever needs another DAW's workflow.

### AI Music Creation

### Datasets

### Community

Music Encoding Initiative (community-driven open-source effort to define a machine-readable music notation encoding), Poly Expression (forum for expressive instruments and controllers).

## Common Mistakes

Treating this snapshot as complete or current — it's a partial capture from one fetch, and several full categories (Synthesizers, Sample Providers, Articles, Research, among others) aren't represented here at all. Refetch the live URL rather than assuming this doc has grown stale gaps filled in.

Citing this doc's contents as if they were vetted, KB-grounded technique recommendations — they're an external community list, not this project's own researched or genre-grounded content.
