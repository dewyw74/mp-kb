# midi/

MIDI programming reference — `patterns/` (bass/drum/melody pattern recipes), `controllers/` (hardware profiles: Akai MPK Mini Plus, Arturia KeyStep 25, Roland MC-303, Evolution MK-425C, plus general CC-mapping standards), `programming/` (piano-roll technique: velocity editing, humanization, groove).

No dedicated schema yet; use YAML frontmatter with at minimum `title` and `tags` until a `midi_schema.json` is added in a later phase.

Consumed primarily by `midi_agent`.
