# mastering/

Mastering-stage reference — `loudness/` (LUFS targets by genre/platform), `dynamics/` (limiting, dynamic range), `streaming/` (platform-specific normalization behavior). Entries conform to `schemas/mixing_schema.json` (shared with `mixing/`; category values like `"loudness"`/`"dynamics"` apply here).

Consumed primarily by `mastering_agent`.
