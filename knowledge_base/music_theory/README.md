# music_theory/

Reference entries on theory building blocks, independent of any single genre — `scales/`, `chords/`, `harmony/`, `melody/`, `rhythm/`. Genre files link back here for their specific scale/progression choices rather than re-explaining theory.

No dedicated schema yet (Phase 1 ships structure only); entries should still use YAML frontmatter with at minimum `title` and `tags` until a `theory_schema.json` is added in a later phase.

Consumed primarily by `composer_agent`.

Filename convention: `snake_case_topic.md` (e.g. `dorian_mode.md`, `secondary_dominants.md`).
