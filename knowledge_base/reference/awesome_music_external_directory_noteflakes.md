# External Directory: noteflakes/awesome-music

[noteflakes/awesome-music](https://github.com/noteflakes/awesome-music) is a second, differently-scoped curated "awesome list" alongside `knowledge_base/reference/awesome_music_production_external_directory.md`. Where that first list is a broad software/hardware/services directory, this one leans toward audio-programming infrastructure: editors, DSP/audio libraries, MIDI tooling, notation, and music-theory-teaching resources, plus its own set of sibling awesome-lists. This entry is a snapshot pointer, not a mirror — the linked repo updates independently of this project, so treat the live URL as the source of truth and refetch it rather than assuming this snapshot stays current.

## When to use this

Consult this doc (or refetch the live URL) when a producer or agent needs an audio-programming library, notation tool, or MIDI-processing utility that isn't documented in depth anywhere else in this knowledge base — this list skews more toward developer/DSP tooling than the first reference entry's end-user software and hardware. Cross-check `awesome_music_production_external_directory.md` first if the need is a DAW, VST, sample provider, or hardware controller, since that list covers those categories more directly.

### Audio Editing

Audacity (free, cross-platform digital audio editor), Ardour (cross-platform DAW emphasizing audio recording), LMMS (cross-platform DAW more oriented toward beatmaking), OpenUtau (free, cross-platform singing voice synthesis framework), snd (sound editor modeled loosely after Emacs), Stargate DAW (cross-platform, all-in-one DAW and plugin suite optimized for EDM production), TuneFlow (a next-gen DAW that helps generate melody, beats, or a whole song while still allowing editing).

### Audio Libraries

audio.js (drop-in JS library for HTML5 `<audio>`), audioFlux (library for audio/music analysis and feature extraction), chord-mark (reference implementation of ChordMark, a text notation format for lyrics/chords/rhythm), howler.js (JS audio library for the modern web), JSyn (audio synthesis API for Java), Librosa (Python package for music/audio analysis, building blocks for music information retrieval), midi.js (framework for web-based MIDI apps), mxml (MusicXML parsing and layout library), PortAudio (free, cross-platform, open-source audio I/O library), and more (list continues past this snapshot).

### Other Awesomeness → Related Lists

This list itself points to 8 sibling awesome-lists worth knowing about if a need falls outside music production specifically: Awesome AI Music Generation, Awesome Audio DSP (audio DSP and plugin-development resources), Awesome Audio Visualization, Awesome Music Listening, **Awesome Music Production (adius/awesome-music-production)** — note this is a *different* GitHub org (`adius`) than the `ad-si/awesome-music-production` fork already documented in this knowledge base's sibling entry; treat them as related but not confirmed identical without checking both — Awesome Music Programming, Awesome Python for Scientific Audio, Awesome sheet music, Awesome WebAudio.

## Common Mistakes

Treating this snapshot as complete or current — 7 of the list's 9 primary categories aren't represented here at all. Refetch the live URL rather than assuming gaps have been filled in.

Citing this doc's contents as if they were vetted, KB-grounded technique recommendations — like its sibling entry, this is an external community list, not this project's own researched or genre-grounded content.

Confusing the `adius/awesome-music-production` related-list entry with the `ad-si/awesome-music-production` list already documented in `awesome_music_production_external_directory.md` — they share a name and general subject but are different repositories; don't assume their contents match without checking.
