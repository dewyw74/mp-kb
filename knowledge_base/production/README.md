# production/

General production craft that isn't tied to one genre — `arrangement/` (song structure techniques), `songwriting/`, `workflow/` (build-order and process guidance), `templates/` (reusable project/session starting points).

`workflow/` entries should conform to `schemas/workflow_schema.json` (`daw: "generic"` where DAW-agnostic). Other subfolders use YAML frontmatter with at minimum `title` and `tags` until dedicated schemas exist.

Consumed primarily by `producer_agent`.
